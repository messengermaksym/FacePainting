# Face Painting Website — Design Specification

> Сайт-візитка для майстра аквагриму у Львові. Single-page landing з модульною архітектурою.

## 1. Project Overview

### Мета
Створити професійний сайт-візитку для майстра аквагриму, яка надає послуги з виїздом по Львову. Сайт має познайомити клієнтів з майстром, показати портфоліо, зорієнтувати по цінах та дати можливість забронювати дату.

### Цільова аудиторія
- **Первинна**: Батьки, які організовують дитячі свята та дні народження
- **Вторинна**: Івент-агенції, організатори корпоративних подій

### Ключова перевага (УТП)
Майстер має професійний художній бекграунд: художня школа → університетська освіта в дизайні (буклети, сайти, графіка) → аквагрим як професійне мистецтво. Це не просто малюнки — це художній підхід з глибокою фаховою базою.

### Локація
Львів, виїзна робота по місту.

## 3. Site Format

**Single-page landing** з якірною навігацією. Вся сторінка — один `index.html`, навігація прокручує до потрібної секції через smooth scroll.

**Мова**: Двомовний (UA 🇺🇦 / EN 🇬🇧) з перемикачем у header. Дефолт — українська. Поточна мова зберігається в `localStorage`.

### Принципи архітектури
- **Один JS-файл = один модуль** з одною відповідальністю
- **i18n через JSON** — всі тексти винесені, легко підтримувати
- **assets відокремлені** — легко замінити placeholder'и на реальні фото
- **dist/ автогенерується** Tailwind CLI — не редагується вручну
- **Відповідає vibe-coding.md**: feature-based, модульна структура без монолітних файлів

## 5. Page Sections (порядок зверху вниз)

### 5.1 Header + Navigation (sticky)

**Елементи:**
- Логотип або назва бренду (текстовий або SVG)
- Горизонтальна навігація (desktop): Про мене · Портфоліо · Послуги · FAQ · Контакти
- Перемикач мови: UA 🇺🇦 / EN 🇬🇧 (toggle або dropdown)
- Hamburger-меню (mobile): slide-in overlay

**Поведінка:**
- Sticky header — прилипає при скролі
- Активний пункт меню підсвічується при скролі (IntersectionObserver)
- Smooth scroll при кліку на навігацію
- Header стає напівпрозорим/з backdrop-blur при скролі

**Модуль**: `navigation.js`

---

### 5.2 Hero Section

**Елементи:**
- Головний заголовок: *«Професійний аквагрим для вашого свята у Львові»*
- Підзаголовок: *«Художній підхід • Безпечні матеріали • Виїзд по Львову»*
- CTA-кнопка: **«Забронювати дату»** → smooth scroll до форми бронювання
- Фонове зображення або декоративні CSS-елементи (paint-splash, м'які градієнти)

**Візуальний стиль:**
- Великий, акцентний перший екран
- Градієнтний overlay на фоновому зображенні (violet → pink, напівпрозорий)
- Paint-splash декоративні форми через CSS (border-radius + rotate)
- Мінімум тексту — максимум впливу

**Модуль**: анімація через `animations.js`

---

### 5.3 Про майстра

**Елементи:**
- Фото майстра за роботою (placeholder)
- Заголовок секції: *«Про майстра»*
- Текстовий блок з історією та підходом
- **Акцент на художній освіті** — виділений блок або timeline:
  1. 🎨 Художня школа — класична художня освіта
  2. 🎓 Університет — професійний дизайн (буклети, сайти, графіка)
  3. ✨ Аквагрим — художній підхід до кожного малюнка

**Візуальний стиль:**
- Двоколонковий layout: фото ліворуч, текст праворуч (desktop)
- Вертикальний timeline з іконками для етапів освіти
- Scroll-reveal анімація появи

---

### 5.4 Портфоліо / Галерея

**Елементи:**
- Фільтри-кнопки: Всі · Дитячі · Тваринки · Казкові · Корпоративні
- Сітка фото (CSS Grid): 12–16 placeholder-слотів
- Lightbox при кліку на фото (збільшення + навігація вліво/вправо)

**Поведінка:**
- Фільтрація з плавною анімацією (fade/scale)
- Адаптивна сітка: 1 колонка (mobile) → 2 (tablet) → 3–4 (desktop)
- Lightbox: ESC для закриття, стрілки для навігації, swipe на mobile

**Модуль**: `gallery.js`

### 5.5 Послуги та ціни

**Картки послуг (3 штуки):**

| Картка | Назва | Що входить |
|--------|-------|-----------|
| 1 | Дитяче свято | Аквагрим для N дітей, вибір малюнків, матеріали |
| 2 | Корпоративний захід | Аквагрим для дорослих та дітей, тематичні малюнки |
| 3 | Індивідуальний аквагрим | Одне обличчя, складний малюнок за запитом |

**Елементи кожної картки:**
- Іконка або ілюстрація
- Назва послуги
- Короткий опис
- Орієнтовна ціна (від X грн)
- CTA: **«Замовити»** → скрол до форми

**Візуальний стиль:**
- Картки з ефектом hover (підняття + тінь)
- Акцентна картка (найпопулярніша) — виділена кольором
- Респонсивність: 1 (mobile) → 3 колонки (desktop)

### 5.6 FAQ (Часті питання)

**Питання (орієнтовні):**
1. Які фарби ви використовуєте? Чи безпечні вони для дітей?
2. Скільки часу тримається аквагрим?
3. З якого віку можна робити аквагрим?
4. Як відбувається замовлення?
5. Чи виїжджаєте за межі Львова?
6. Скільки часу потрібно на один малюнок?

**Поведінка:**
- Accordion: клік на питання розгортає відповідь
- Одночасно відкрите лише одне питання (при кліку на нове — попереднє закривається)
- Плавна анімація розгортання (CSS transition на max-height)

**Модуль**: окрема логіка в `app.js` або виділений accordion через `data-` атрибути

### 5.7 Форма бронювання

**Поля форми:**

| Поле | Тип | Обов'язкове | Валідація |
|------|-----|-------------|-----------|
| Ім'я | text | ✅ | min 2 символи |
| Телефон | tel | ✅ | формат UA (+380...) |
| Email | email | ❌ | формат email |
| Тип заходу | select | ✅ | Дитяче свято / Корпоратив / Інше |
| Бажана дата | date | ✅ | не раніше завтра |
| Кількість дітей | number | ❌ | 1–100 |
| Коментар | textarea | ❌ | max 500 символів |

**Поведінка:**
- Inline-валідація (помилки з'являються під полем при blur)
- Submit → Formspree API (POST)
- Success: анімоване повідомлення «Дякуємо! Ми зв'яжемось з вами найближчим часом»
- Error: повідомлення про помилку з пропозицією спробувати ще раз

**Модуль**: `form.js`

## 7. i18n Architecture

### Підхід
- HTML-елементи з `data-i18n="key"` атрибутом
- JSON-файли з перекладами: `src/i18n/uk.json`, `src/i18n/en.json`
- JS-модуль `i18n.js`:
  - Завантажує JSON при старті
  - Підставляє тексти за ключами
  - Зберігає вибір мови в `localStorage`
  - Слухає клік на перемикачі мови

### Приклад HTML:
```html
<h1 data-i18n="hero.title">Професійний аквагрим...</h1>
```

### Приклад JSON (uk.json):
```json
{
  "hero": {
    "title": "Професійний аквагрим для вашого свята у Львові",
    "subtitle": "Художній підхід • Безпечні матеріали • Виїзд по Львову",
    "cta": "Забронювати дату"
  },
  "nav": {
    "about": "Про мене",
    "portfolio": "Портфоліо",
    "services": "Послуги",
    "faq": "FAQ",
    "contact": "Контакти"
  }
}
```

## 8. Deployment

- Автодеплой з GitHub repository
- Вбудований HTTPS (SSL)

## 10. Performance & SEO

### SEO
- Семантичний HTML5: `<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`
- Один `<h1>` на сторінці (в Hero)
- `<title>`: «Аквагрим Львів — Професійний аквагрим на свята | [Ім'я майстра]»
- `<meta description>`: Опис послуг для пошукових систем
- `lang="uk"` (перемикається через i18n)
- Open Graph теги для соцмереж
- Alt-тексти на всіх зображеннях

### Performance
- Tailwind CSS з purge (tree-shaking невикористаних класів)
- Зображення: lazy loading (`loading="lazy"`)
- Мінімізований CSS (--minify в build)
- JS-файли підключені з `defer`
- Шрифти: `font-display: swap` для Google Fonts

## 11. Out of Scope (YAGNI)

Ці фічі **НЕ** входять в поточну версію:
- Блог / Новини
- Онлайн-оплата
- Календар зайнятості (реальний)
- Адмін-панель
- CMS / редагування контенту через інтерфейс
- Відгуки клієнтів (не обрано користувачем)
- Блок «Як це працює» (не обрано користувачем)
- Карта Google Maps (можливе розширення)
